# Send emmails with Office 365

## Prerequisites
* PHP 8.1
* Composer

<br>

## Installation

```
composer install
```
In `src/EmailSenderOffice365` replace USERNAME and PASSWORD by credential provided by Microsoft.

<br>

## Usage

```
php index.php
```
